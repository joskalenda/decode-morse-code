# Decode Morse
> This project uses Ruby to decode morse code.

## Built With
- Ruby
- git

## Getting Started
To get a local copy up and running follow these simple example steps.
Clone the project by using terminal:
```
git clone https://github.com/farouqdaif1/Decode.git
ruby decode.rb
```
### Prerequisites
-Ruby
